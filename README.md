# PAMPERS Beer Bransh Mashup

## Kenneth Rose

### This is a brand mashup activity I did involving Pampers and Heineken.
### I merged the branding of Pampers into that of Heineken.
### The goal was to make it at first glance appear to be Heineken until people looked closer.

## Site built using HTML, CSS, and JavaScript.
## The site is dynamic and should work on screen 350px or larger.
### Site tested compatible in Firefox, Chrome, and Safari.

## To Open:
1. Clone repo
2. Launch `<index.html>` in any modern web browser.

Images are mix of [stock images](https://unsplash.com/), and created by me.
3D renderings completed in [Cinema 4D](https://www.maxon.net/en/cinema-4d).
Images, and Videos created using the [Adobe CC Suite](https://www.adobe.com/ca/).
